# Bilibili 弹幕下载器

下载 B 站视频弹幕的命令行工具，支持单个视频和多 P 视频的弹幕下载。

## 功能特性

- ✅ 支持通过 BV 号下载弹幕
- ✅ 自动获取视频标题作为文件名
- ✅ 支持多 P 视频分集下载
- ✅ 支持多 P 弹幕合并下载
- ✅ 输出标准 B 站 XML 格式弹幕文件
- ✅ 油猴脚本版 — 在 B 站页面内一键下载

## 版本说明

| 版本 | 文件 | 说明 |
|:---|:---|:---|
| v1 基础版 | `src/get-danmaku.js` | 命令行工具，适用于单 P 视频 |
| v2 进阶版 | `src/get-danmaku-v2.js` | 命令行工具，支持多 P + 合并下载 |
| v3 油猴版 | `src/get-danmaku-v3.user.js` | 浏览器油猴脚本，在 B 站页面内操作 |

## 使用方法

### v1 基础版 (命令行 · 单 P 视频)

**环境要求**: Node.js >= 14.0

```bash
node src/get-danmaku.js BV1xx411c7mD
```

### v2 进阶版 (命令行 · 多 P 视频)

**环境要求**: Node.js >= 14.0

**分 P 下载 (默认模式)**
```bash
node src/get-danmaku-v2.js BV1xx411c7mD
```

**合并下载**
```bash
node src/get-danmaku-v2.js BV1xx411c7mD --merge
# 或
node src/get-danmaku-v2.js BV1xx411c7mD -m
```

### v3 油猴版 (浏览器插件)

在浏览器中直接使用，无需 Node.js 环境。

**安装步骤：**

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 点击下方链接安装脚本（或在 Tampermonkey 中手动新建脚本并粘贴代码）：
   - [从 GitHub 安装](../../raw/main/src/get-danmaku-v3.user.js)
3. 打开任意 B 站页面，右下角出现粉色悬浮按钮，点击打开下载面板

**功能一：手动下载（视频页）**
- 🎯 自动识别当前视频 BV 号（需在视频页手动刷新后使用）
- ⬇️ 支持逐 P 下载和合并下载
- 📈 实时下载进度反馈

**功能二：收藏夹自动轮询**
- 📂 配置一个 B 站收藏夹 ID，脚本每 6 小时自动下载其中所有视频的弹幕
- 收藏夹 ID 获取方式：打开收藏夹页面，URL 中 `fid=` 后面的数字即为 ID
- 支持开关控制、立即执行
- 下载完成后发送桌面通知

**多标签页与防重复机制：**
- **调度锁（Lease-based）**：主控标签页每 5 分钟续期一个时间戳，避免多标签页同时成为主控
- **轮询互斥锁**：通过 `GM_setValue` 的 `POLL_RUNNING` 键做全局互斥，防止并发执行轮询

> ⚠️ 注意：自动轮询需要至少保持一个 B 站标签页打开。关闭浏览器后轮询将暂停，重新打开 B 站后会自动恢复。自动轮询的日志文件（`轮询日志_年-月-日.txt`）会与弹幕 XML 一同下载至浏览器的默认下载目录中。

**功能三：下载日志**
- 📋 记录每次下载的时间、视频标题、弹幕数量和状态
- 可随时查看日志、一键清空，最多保留 500 条记录

**功能四：性能诊断面板**

面板底部有一个诊断栏，帮助确认脚本对性能的实际影响：

```
标签角色：主控    ← 所有标签页中只有一个是 主控，其余为 待机
后台唤醒：3 次    ← 定时器自页面加载以来触发的次数
```

- **主控**：负责实际的 6 小时轮询调度，每 5 分钟检查一次是否到时间
- **待机**：仅监视主控标签页是否关闭，关闭后自动接管；行为与主控完全一致（也是每 5 分钟唤醒一次）
- 无论打开多少个 B 站标签页，脚本整体每 5 分钟只做一次轻量的内存读取，不发起任何网络请求

## 输出文件

- 基础版: `视频标题_BV号.xml`
- 进阶版 / 油猴版 (分 P): `视频标题_P1_分集名_BV号.xml`
- 进阶版 / 油猴版 (合并): `视频标题_[全集合并]_BV号.xml`
